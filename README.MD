# Integracion Banco Ganadero

Integracion de Servisofts con el Banco Ganadero a travez de una VPN IPSec iKev2.
Para esta integracion estamos utilizando un firewalld de la marca hillstone `sg-6000`.


## Installation

- `hilstone sg-6000`
- `IP fija`
